> Daftar pustaka disusun menggunakan format APA 7th Edition.
>
> ### Contoh Format
>
> **Buku:**
> ```
> Nama Penulis. (Tahun). *Judul Buku*. Penerbit.
> ```
>
> **Jurnal:**
> ```
> Nama Penulis. (Tahun). Judul artikel. *Nama Jurnal*, *Volume*(Nomor), Halaman. DOI
> ```
>
> **Sumber Online:**
> ```
> Nama Penulis. (Tahun). *Judul Halaman*. Nama Website. URL
> ```
